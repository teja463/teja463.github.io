# Vim basic shortcuts

| Command | Description |
| ------ | ------ |
| yy | Copies line |
| p | Pastes line |
| Ctrl + u | Page UP |
| Ctrl + d | Page down |
|  j | Down arrow |
|  k | Up arrow |
|  l  | Right arrow |
|  h  | Left arrow |
| 10G | Go to 10th line |
|  w | Go to next word |
|  b | Backward to next word |
|  u |Undo |
| Ctrl + r | Redo |
